# Calculate average percent difference between two sensors

Calculate average percent difference between two sensors

## Usage

``` r
avg_pct_diff(data, col1, col2)
```

## Arguments

- data:

  Dataset with wide pivoted sensor/pollutant columns

- col1:

  Name of column of sample measurements of sensor 1

- col2:

  Name of column of sample measurements of sensor 2

## Value

Average percent difference of two sensors
