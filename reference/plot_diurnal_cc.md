# Diurnal Plot - Community Call

Diurnal Plot - Community Call

## Usage

``` r
plot_diurnal_cc(data, site, start, end, statistic, title)
```

## Arguments

- data:

  Quant data for plot

- site:

  A string Quant sensor name

- start:

  Start date

- end:

  End date

- statistic:

  A string "Median" or "Mean"

- title:

  Plot title

## Value

Diurnal ggplot
