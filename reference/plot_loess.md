# Loess Regression Plot for any Pollutant & Monitor(s)

Loess Regression Plot for any Pollutant & Monitor(s)

## Usage

``` r
plot_loess(data, x, y, title, xlab, ylab, caption = NULL)
```

## Arguments

- data:

  Pollutant Dataset. Must be in 'Date \| Quant \| BAM' form.

- x:

  Independent variable

- y:

  Dependent variable

- title:

  Title of the plot.

- xlab:

  X axis label.

- ylab:

  Y axis label.

- caption:

  Equation caption.

## Value

A Loess Smooth Line ggplot
