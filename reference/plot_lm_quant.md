# Linear Model Plot for any Pollutant & Monitor(s)

Linear Model Plot for any Pollutant & Monitor(s)

## Usage

``` r
plot_lm_quant(data, title, ind, dep, pollutant)
```

## Arguments

- data:

  Pollutant Dataset. Must be in 'Date \| Monitor \| Pollutant' form.

- title:

  Title of the plot.

- ind:

  Independent variable.

- dep:

  Dependent variable.

- pollutant:

  Pollutant to compare.

## Value

A Linear Model ggplot
