# Median Regression Plot for any Pollutant & Monitor(s)

Median Regression Plot for any Pollutant & Monitor(s)

## Usage

``` r
plot_rq(data, x, y, title, xlab, ylab, caption = NULL)
```

## Arguments

- data:

  Pollutant Dataset. Must be in 'Date \| Quant \| BAM' form.

- title:

  Title of the plot.

- xlab:

  X axis label.

- ylab:

  Y axis label.

- caption:

  Equation caption.

## Value

A Median Regression ggplot
