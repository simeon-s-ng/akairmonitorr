# Reads an Agilaire Regulatory Flagged Data Export

Reads an Agilaire Regulatory Flagged Data Export

## Usage

``` r
read_regulatory_flagged(file, site)
```

## Arguments

- file:

  The path to the file.

- site:

  The Regulatory site name as a string. This should match the site name
  exported from AA.

## Value

Agilaire data in tibble form.
